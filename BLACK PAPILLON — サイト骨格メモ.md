WORKS
目的（1行）：最初の3秒で「上手い」を証明して、言葉の前に信頼させる
入れる情報
縦位置写真（人体カーブに沿う構図）
最初の3枚は強く（占有率高め・“殴る”）
4枚目以降は余白を増やして静かに流す
キャプションは最小（部位だけ or 0）
見せ方の文法：最初の3枚で殴る → 残りは沈黙で格を保つ
HEALED（核）
目的：「時間が経つと汚くなる？」を写真1枚で潰す
入れる情報
healed写真（可能なら“直後 vs healed”）
断言コピー（1文1行）
仕上がりで、選ぶ。
時間が経っても、線が崩れにくい。
治癒後を前提に設計する。
“なぜ崩れにくいか” 1行（技術語なしの平易な理由）
見せ方の文法：写真が先、コピーが後（逆にすると説明臭）
STYLE
目的：「自分が頼めるか」を3秒で判断させる
入れる情報（各スタイル）
Fine line：線の精度。細さより、崩れにくさ。
Black & Grey：黒の階調。汚れず、沈む。
Coverup：消すのではなく、更新する。
各：1見出し＋1文＋縦写真
見せ方の文法：3カラムカード禁止。縦にスタックして“川”で流す
CARE（PROCESS + SAFETY 統合）
目的：「初めてでも怖くない」と「来てほしい客層のフィルタ」を同時にやる
入れる情報（dl形式／箇条書き圧縮）
相談 → デザイン確認 → 当日施術 → アフターケア
当日の目安（所要は相談で提示／体調不良リスケOK）
衛生（使い捨て／施術ごとの交換…など具体動作で）
条件（18+、アレルギー等は事前申告）
見せ方の文法：硬くていい。硬い方が信頼。“ふわっと衛生徹底”は禁止（固有動作で）
BOOKING
目的：質の良い問い合わせだけ集め、DM往復を先に潰す
入力項目（最小7）
名前
連絡先（LINE / IG / Mail のどれか1つ）
希望スタイル（Fine line / B&G / Coverup / まだ不明）
部位（ざっくりOK）
サイズ感（手のひら以下 / A4 / それ以上 / 不明）
予算目安（〜3万 / 3〜5万 / 5万以上 / 相談）
参考画像 or メモ（任意）
表示注意（3行で十分）
18歳未満不可
返信目安：2〜3営業日
最初のメッセージは短くて大丈夫
見せ方の文法：CTAは上品に。“相談歓迎”の言い方で雑DMを増やさない
FOOTER（ACCESS統合）
目的：BOOKINGの余韻を切らずに“着地”
入れる情報
営業時間 / 地図 / SNS / 連絡先（コンパクト）
“BOOKING → footer情報” で自然に終わる
Color（最低4色 / baseは黒）確定

BLACK PAPILLONは **4色＋派生（line/muted）**で十分。

Base：Black（背景）
Sub：Petrol（深度）
Accent：Oxblood（噛み）
Text：Off-white（白）
Typography（h1=SVG固定 / h2+p をコードで）
h2（英字・ラベル・UI）：Space Grotesk（硬い精密さ）
p（本文・日本語）：IBM Plex Sans JP（清潔で読みやすい）
